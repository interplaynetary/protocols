## Documentation Structure

### Core Concepts
- **ValueFlows**: Based on REA accounting and ValueFlows ontology
- **Holochain**: Distributed application framework for data storage and networking
- **GraphQL Interface**: JavaScript adapter library for API access

### Key Components
- **Agents**: People and organizations in the economic network
- **Events**: Observable movements of resources and economic flows
- **Resources**: Economic resources tracked in the system
- **Processes**: Business processes and workflows
- **Agreements**: Contracts and commitments between participants

### Technical Documentation
- **Zomes**: Holochain modules for core functionality
- **DNA**: Distributed network application definition
- **GraphQL Schema**: API interface specification
- **Client Libraries**: JavaScript/TypeScript integration
