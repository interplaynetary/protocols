# Changelog

## [1.0.0] - 2024-12-06

### Added
- Initial release of `@playnet/free-association` protocol package
- Pure TypeScript implementation of Free Association protocol
- Core allocation algorithm with recognition-based distribution
- ITC (Interval Tree Clocks) for causality tracking
- Slot-based matching system
- Compliance and eligibility filters using JsonLogic
- Tree operations for recognition hierarchies
- Collective coordination support
- Attribute recognition system
- Full TypeScript type definitions

### Features
- **Framework-agnostic**: Works in Node.js, browsers, and any JavaScript environment
- **Zero dependencies** (except zod and json-logic-js)
- **Type-safe**: Full TypeScript support with exported types
- **Modular**: Import specific modules as needed
- **Well-documented**: Comprehensive README and inline documentation

### Exports
- Main entry: `@playnet/free-association`
- Subpath exports for all modules
- ITC causality tracking
- Filter utilities
- Attribute system
- Collective coordination
